# Vertical Shooter

This is a small vertical scrolling shooter built with **Godot 4.4.1**.

## Running the Game
1. Install [Godot 4.4.1](https://godotengine.org/).
2. Open `project.godot` with Godot 4.4.1 and run the project.

### Controls
- **WASD** or **Arrow Keys**: Move the ship
- **Space**: Shoot
- **Escape**: Pause
